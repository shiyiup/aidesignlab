# aidesignlab.cc

AI Design Lab — Practical AI design tutorials, tool reviews, and prompt templates.

## Quick Start

Open `index.html` directly in a browser, or serve locally:

```bash
npx serve .
```

## Deploy to Vercel (Free)

1. Push to GitHub:
   ```bash
   cd aidesignlab
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/aidesignlab.git
   git push -u origin main
   ```

2. Go to [vercel.com](https://vercel.com), sign in with GitHub, click "New Project", select this repo, click Deploy.

3. Connect your domain `aidesignlab.cc` in Vercel Settings → Domains.

## Domain Setup

1. Buy domain at Namecheap/Cloudflare
2. In Vercel → Domains, add `aidesignlab.cc`
3. Vercel will give you nameservers or DNS records to configure
4. Update nameservers at your domain registrar
